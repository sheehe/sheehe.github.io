口袋斗罗大陆 - 成神活动任务明细

目标总进度 2000点

说明：

  • 本次活动总目标为 2000点 进度。
  • 有部分进度是相对固定或付费获取的：
    • 七天累积登录：70点
    • 七天累积购买2500钻石：70点
    • 七天累积购买68元礼包：350点 (每日购买，7天共计 7 * 50 = 350点)
    这部分固定进度共计 70 + 70 + 350 = 490点。
  • 因此，通过完成活动任务所需达成的目标进度为 2000点（总目标）- 490点（固定获取）= 1510点。
  • 每个任务要求达成后获得 10点 进度。
  • 本表列出了所有可通过完成任务获取的进度明细，其最高总进度为 1530点（包含第五天转盘抽奖的100点）。我们计算1510点任务目标时，不考虑第五天转盘抽奖的100点进度。
  • 基于表中列出的任务（排除转盘100点），任务最高可获取 1530点 - 100点 = 1430点。而我们需要通过任务获取 1510点。
  • 这意味着表中列出的任务不足以单独达成1510点的任务目标，需要额外从表格中未列出的招募、魂环等资源消耗型任务的更高档次中获取 1510点 - 1430点 = 80点（即8项更高要求）。

表格的关键信息总结：

  • 详细任务要求表： 提供每日各类任务的完整进度档次要求（例如御堂等级、药材消耗等）以及每个任务档次对应的目标数值。可以查看每项任务的详细分解和单个任务的最高进度。
  • 简略任务及进度表： 快速概览每日所有任务名称，以及每个任务理论上能拿满的最高进度。并汇总了每日和累积的最高任务总进度。
  • 请注意，这两张表展示的累积最高任务总进度为1530点（含转盘）/ 1430点（不含转盘），低于达成2000点总目标所需的1510点任务进度，缺口80点需依赖未列出的更高档次任务。

达成总进度2000点的示例 (基于任务进度1510点 + 固定490点)

以下示例提供一种达成总进度2000点的方式。您需要从所有活动任务中累积获取1510点进度，同时获取固定的490点。

如说明中所述，本表列出的任务最高可提供1430点（不含转盘）。达成1510点的任务目标，需要额外从表格中未列出的招募、魂环等资源消耗型任务的更高档次中获取 80点（即完成8项更高要求）。

  1. 示例：完成所有列出的任务要求 (不含转盘)，并额外完成8项更高要求。
    一种达成方式是：
    • 获得固定的登录、购买钻石、购买68礼包进度：490点。
    • 完成本表中列出的所有任务要求（排除第五天转盘的10项要求）：共计1430点。
    • 额外完成表格中未列出的招募、魂环等资源消耗型任务的更高档次要求：再获得80点进度（例如，如果招募任务有更高的21, 22, ..., 28档次，完成这8档即可）。

    总进度： 1430点 (表列任务) + 80点 (更高档次任务) + 490点 (固定) = 2000点。
